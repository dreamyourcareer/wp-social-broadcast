== Description ==

This plugin will provide your site with a unique possibility to viral spreading the news you provde on social networks such as Twitter, Facebook, Linkedin, Google, Yahoo, Hotmail and more.

Full documentation available on [our site](http://www.WPSocialbroadcast.com)

= Currently Supported Providers =

Gmail, Facebook, Twitter, Linkedin, Yahoo, Live, Foursquare.

= Features =

* Different API limits
* Sidebar widget for sidebars
* HTML emails
* Predefined invitation messages
* Custom CSS

= Premium version =

* Unlimited usage not limited for 30 days as in Free version
* GMAIL SUPPORT
* Redirect users after they send broadcast
* Free Support

Get the premium version on [http://www.WPSocialbroadcast.com]

== Installation ==

1. In your WordPress dashboard, go to menu Plugins > Add New
2. Search for `WP Social Broadcast` or upload WP Social Broadcast zip file
3. Click to install
4. Activate the plugin
5. A new menu `WP Social Broadcast` will appear in your Admin
6. Enjoy the plugin

Read the [documentation](http://www.WPSocialbroadcast.com/docs/) and configure the desired providers.
